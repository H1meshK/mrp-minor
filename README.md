# Web-Based MRP System for Inventory & Production Planning

A web-based **Material Requirements Planning (MRP)** system designed to streamline **inventory control, BOM management, and production scheduling** for small-to-medium manufacturing environments. The system emphasizes **operational simplicity, real-time visibility, and decision support**, avoiding the complexity of traditional ERP solutions.
